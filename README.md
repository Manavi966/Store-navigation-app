# 🛒 SmartMart Navigator

A smart store navigation system that helps users quickly find products inside a store using an interactive map and recommendations.

---

## 🚀 Live Demo
👉 https://store-navigation-app.vercel.app

---

## 💡 Problem Statement
Finding products in large supermarkets is time-consuming and confusing.

---

## ✅ Solution
SmartMart Navigator provides:
- 📍 Product location inside store
- 🗺️ Interactive store map
- 🔍 Easy product search
- 📷 QR-based navigation (if applicable)

---

## 🛠️ Tech Stack
- Next.js
- React
- TypeScript
- Tailwind CSS

---
